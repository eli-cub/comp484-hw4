On this page you’ll find a quick reference to common HTML terms and a table of frequently used named character entities.

HTML Terms

Attribute
An HTML element property that provides additional information or functionality for an element. Attributes appear as a name/value pair with the value contained in quotation marks.
Block-Level Element
HTML tags that create blocks of content within the document such as headings and paragraphs. Block-level elements typically stack one on top of each other based on the order they appear in the code.
Child Element
Any HTML element that is contained inside another HTML element. Child element tags must close before their parent element tags.
Closing Tag
Appears after an element’s content and indicates the end of an element. Identical to the opening tag with the addition of a forward slash (/).
Comment
Reference information added to the code. May include an overview of complex code, reminders, or communications within a team.
CMS
Short for Content Management System. Refers to systems designed to allow users to create and maintain content (e.g., Joomla!, WordPress).
CSS
Cascading Style Sheets, a presentation language used to control the layout and appearance of web pages.
Doctype
Short for Document Type Declaration. Appears at the top of all HTML documents to trigger standards mode and identify the version of HTML.
DOM
The Document Object Model, a representation of documents like HTML, where tags are represented by “nodes.” JavaScript uses the DOM to access or modify page content.
Elements
HTML elements consist of an HTML tag and the content it encloses. Often confused with “tags,” which refer only to the markup itself.
FTP
Stands for File Transfer Protocol, used to transfer large files from one server to another.
HTTP
Hypertext Transfer Protocol, used to format and transmit HTML pages across web servers.
Inline-Level Elements
Elements that appear within the normal flow of text and do not start new lines, unlike block-level elements.
JavaScript
Scripting language developed by Netscape, used to control functionality and behavior on the web.
Metadata
Additional information about the document or its content, typically stored in meta tags.
MIME Type
Stands for Multipurpose Internet Mail Extension. Tells user agents what type of file or data is being transmitted.
Parent Element
An HTML element that contains another element. An element can be both a parent and a child at the same time.
Replaced Element
An HTML element whose content is replaced by something else, such as images, video, or other embedded objects.
Tag
Markup characters that indicate what type of content appears on the page. Most tags have both an opening and closing form.
URL
Uniform Resource Locator, the address of a document or resource on the web.
W3C
World Wide Web Consortium, the standards body responsible for authoring the HTML specification.
